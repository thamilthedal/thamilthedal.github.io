foamToFluent2D CLI

I built a CLI tool that converts 2.5D meshes generated via OpenFOAM blockmesh into 2D meshes for ANSYS Fluent Simulations.

  • Written in Python and Works on structured meshes so far.

  • Direct Conversion from Polymesh to 2D Fluent Mesh.

  • Conversion faster than OpenFOAM in-built foamToFluentMesh utility. More info on development journey can be found heere.

  • Check out the Github repository here.

residual_hpc CLI

I built a CLI tool that can monitor Residuals and Report files for ANSYS Fluent simulations running through a HPC Cluster.

  • Python Utility auto updates residual monitor of running simulations at a regular frequency.

  • Can be installed and started with just SSH Credentials to be entered only once at the beginning and encrypted securely.

  • Report Files can be monitor separately for the running simulations in addition to the residue.

  • Check out the Github repository here.

supercritical CLI

A CLI handling full-fledged simulation workflow right from modelling and meshing, case creation, job handling in Cluster, monitoring, postprocessing and report generation.

  • Full-fledged Python module that can be extended to any usecase for running simulations in ANSYS Fluent with and without UDFs also.

  • Modelling and Meshing are automated through blockmesh in OpenFOAM and separate mesh-dependency utility is also present.

  • ANSYS Fluent v21R2 and OpenFOAM are essential to run the package.

  • In-built terminal based residual monitor, Separate Post-processing utility and Variable property generation module.

  • Separate commands for Job handling, submitting, deleting, clearing up files in Cluster.

  • Contact me for a demo or building one for your usecases.
